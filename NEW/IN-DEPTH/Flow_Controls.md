# Flow Control

### `if` / `else if` / `else`

##### Examples

1. Regular way
>```
if(cond) {
    state1;
} else if(cond2) {
    state2;
} else {
    state3;
};
``` 

2. Shorter way
> `false ? a : b`

### `switch`

##### Example

> 
`switch (day) {`    
&nbsp;&nbsp;
`    case MON:`  
&nbsp;&nbsp;&nbsp;&nbsp;
`        statement;`   
&nbsp;&nbsp;&nbsp;&nbsp;
`        break;`  
&nbsp;&nbsp;
`    case TUE:`  
&nbsp;&nbsp;&nbsp;&nbsp;
`        statement;`   
&nbsp;&nbsp;&nbsp; &nbsp;
`        break;`  
&nbsp;&nbsp;
`    case WED:`  
&nbsp;&nbsp;&nbsp;&nbsp;
`        statement;`    
&nbsp;&nbsp;&nbsp;&nbsp;
`        break;`  
&nbsp;&nbsp;
`    default:`  
&nbsp;&nbsp;&nbsp;&nbsp;
`        statement;`  
`};`
